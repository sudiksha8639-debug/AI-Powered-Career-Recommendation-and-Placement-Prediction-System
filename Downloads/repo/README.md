# 📖 Project Overview

The **AI-Powered Career Recommendation and Placement Prediction System** is a supervised Machine Learning project that predicts whether a student is likely to be placed based on academic performance and employability-related factors.

In addition to placement prediction, the project provides career guidance by recommending suitable technical domains and learning paths based on the prediction results.

This project demonstrates a complete Machine Learning workflow including:

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Hyperparameter Tuning
- Model Evaluation
- Career Recommendation Module

- SSC Board

- HSC Stream
- Degree Percentage
- Degree Type
- Work Experience
- Employability Test Score
- MBA Percentage
- MBA Specialization

Target Variable:

**Placement Status**

- Placed
- Not Placed

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sudiksha8639-debug/AI-Powered-Career-Recommendation-and-Placement-Prediction-System.git
```

Move into the project directory

```bash
cd AI-Powered-Career-Recommendation-and-Placement-Prediction-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook using Google Colab or Jupyter Notebook.
